Project Portfolio Overview

Each project below includes real deliverables,technology stacks and practical learnings from working in collaborative team environments.

1.To-Do-List

Project Details

A web-based application that helps users organize and manage daily tasks efficiently. Users can add, update, delete, and track the status of their tasks.

Features:

  • Add new tasks
  • Edit existing tasks
  • Delete tasks
  • Mark tasks as completed
  • Task filtering (Completed/Pending)
  • User-friendly interface

Technologies Used:

HTML,CSS,JavaScript,Local Storage / MySQL

Challenges:

Storing task data efficiently
Managing task status updates
Preventing data loss on page refresh
Creating a responsive user interface

Future Enhancements:

User authentication,Task reminders and notifications,Priority-based,task management,Cloud synchronization,Mobile application support

Future Enhancement:

3.Mini Blogging Website

Project Details:

A Mini Blogging Website is a web application that allows users to create, publish, edit, and delete blog posts. Users can share their thoughts, ideas, and experiences through blogs while reading posts from other users. The platform provides a simple and user-friendly environment for content creation and management.

Features:

  • User Registration and Login
  • Create New Blog Posts
  • Edit Existing Blogs

Technologies used:

  1. Frontend
    • HTML
    • CSS
    • JavaScript
  2. Backend
    • Flask
  3. Database
    • MYSQL

Challenges:

Managing user authentication securely, Preventing unauthorized access, Designing a responsive layout, Handling blog CRUD operations efficiently

Future Enhancement:

Like and Comment System, Rich Text Editor, Blog Sharing on Social Media, AI-Based Content Suggestions
